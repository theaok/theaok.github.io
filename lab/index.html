Welcome to computational social science lab!

The lab is meant for scientific computing, and espcially general student/faculty collaboration in computational research, so called Computational Social Science (CSS) or Social Data Science (http://science.sciencemag.org/content/323/5915/721). It is also meant for workshops and small classes (eight terminals including one for instructor). As of 2018, these are some of the fastest, latest and greatest computers (short of parallel computing) using 2K monitors for productive scientific computing.

The philosophy behind the lab is to encourage the best practices for scientific computing (http://journals.plos.org/plosbiology/article?id=10.1371/journal.pbio.1001745), which in general means producing reproducible research by writing computer code and avoiding pointing-and-clicking.

Notably, the lab is open-source: it uses free community-developed software (https://www.linuxfoundation.org, https://www.fsf.org).

    • using the lab

    Either right click on the desktop to access main software or press F12 and type name of program to run. To toggle between windows press Alt-Tab: there is no bottom or upper menu bar like in windows or in mac to point-and-click on programs. Likewise, there is no point-and-click windows manager. To operate on files (move, copy, etc): just use bash in terminal: http://tldp.org/LDP/abs/html/basic.html. To mimic two monitors, there are two desktops: desktop 1 (default) is accessed by pressing F1, and desktop 2 is accessed by pressing F2; you can also toggle between them using mouse scroll on desktop.

    The main computer (father) is connected to the TV or you can connect your laptop via HDMI using extra HDMI cable, there is also remote to use the TV.
