short GIS course


No prerequisites, but needs to be comfortable using a computer--it'is mostly about using (GIS) software.

GIS is useful in all fields, because no matter what you study, it always takes place somewhere and place matters. This is an introductory mapping workshop. This workshop is very applied with many tutorials and examples using modern GIS software (QGIS). You will bring your own data and/or I will help you find data matching your interests. Email me ahead of time!
